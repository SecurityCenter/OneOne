# OneOne
### 简介
OneOne是一个PHP开源的内容管理框架, 基于最新Laravel 5.5 版本开发, 提供更高效, 快速, 更安全的WEB应用开发体验, 功能强大超乎想象, 这一切都是为了让开发工作变得简单的.
